## E. Полиш и «сухие»/редизайн модули (нужны твои решения по дизайну)
- **Тренировки**: `videoUrl` у всех null, нет голосовых подсказок (видео — Ф2/CDN). Решение: хостить видео/GIF или отложить.
- **Медитации**: [x] редактор «своя сессия» сделан (`cfc1f09`). Контент: расширяем 5→10 сессий + **поза** к каждой (контент пишет ассистент, пользователь утверждает; поток — поза/описание читаются ДО старта, технику приложение ведёт по таймеру ВО время). **Аудио (РЕШЕНО, приоритет выше видео):** для медитации важны **голос + музыка**, не видео. Голос реализуем **без хостинга через on-device TTS** (`flutter_tts`) — читает шаги вслух (решает «глаза закрыты»); музыка — встроенные ambient-лупы. Видео — отложено (Ф2/CDN). Само аудио — отдельная задача после контента.
- **Осанка**: статичный текст, без гайд-таймера. Можно переиспользовать таймер дыхания/медитации.
- **Дыхание**: движок чистый и data-driven (своя техника — тривиально по данным), но нет UI-редактора/persist. Решение: UX билдера фаз.
- **Стакан воды**: два CustomPaint-стакана (`water_fullscreen_screen.dart`); **хардкод-английские строки** (`+$ml ml`, `health_screen.dart:230`). Решение: новый визуал. **+ Пересмотреть инкременты добавления:** карточка «Здоровье» — кнопки **+250/+500 мл** (`health_screen.dart:283/291`), а полноэкранный вид — **150/200/250/350** (`water_fullscreen_screen.dart:78`) → два разных набора, выглядит случайно. **РЕШЕНО (пользователь):** один канонический набор объёмов; **кнопки карточки = подмножество набора полноэкранного** (значения всегда согласованы). Механика: полноэкранный `/water` показывает весь набор + **«своё количество»**; компактная карточка «Здоровье» показывает 2 самые ходовые из того же набора, а за остальным/кастомом — тап открывает полноэкранный. Предложение по числам (финал в дизайн-фазе): набор **200/300/500** (+«своё»), на карточке **300/500**; опц. «размер моего стакана» в настройках подменяет среднюю кнопку. NB: сейчас 2 экрана с РАЗНЫМИ наборами (карточка 250/500 vs полноэкранный 150/200/250/350, пересечение только 250) — унифицировать.
- **Экран здоровья**: 9 плоских одинаковых плиток — сгруппировать/оформить. Решение: группировка/визуал.
- **Отчёты/Wrapped**: нет кольца/стрика/кнопки «Поделиться» (есть в SPEC). Решение: лейаут + share-картинка.
- **Долгосрочные цели**: функциональны, «сухость» = простые списки (по твоим словам «может и к лучшему») — полиш опционально.
