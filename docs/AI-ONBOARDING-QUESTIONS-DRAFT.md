## 2. Черновик вопросов брейндампа (на утверждение)

Экран после регистрации: **одно большое поле** (голос ИЛИ текст — переиспользуем `VoiceTextField`), а
вопросы-подсказки идут **рядом/сверху по одному**, чтобы мысль не терялась. Человек может ответить сплошным
рассказом — ИИ сам разложит. Вопросы = только подсказки, не обязательные поля.

**EN (ships first) / RU:**

1. **What do you do day-to-day?** (study, work, a project, sport…)
   *Чем занимаешься сейчас? (учёба, работа, проект, спорт…)*
2. **What's important not to drop in the next few days?**
   *Что важно не упустить в ближайшие дни?*
3. **Any deadlines or fixed dates coming up?**
   *Есть дедлайны или даты, к которым нужно успеть?*
4. **What do you keep putting off?**
   *Что чаще всего откладываешь?*
5. **How do your days usually look — packed or spread out?**
   *Дни обычно плотные или растянутые?*
6. *(опц., покажем только если релевантно)* **Want me to help track food / water / sleep too?**
   *Помочь ещё и с питанием / водой / сном?*

**UX-обыгрыш (беру на себя, критерий «удобно, мысль не теряется»):**
- Одно поле-«холст» + чип-строка/список из 6 подсказок над ним; подсказка, на которую человек ответил,
  тихо гаснет (галочка) — виден прогресс, но ничего не обязательно.
- Микрофон крупный (mobile), на web — текст (STT скрыт на web, как везде).
- Кнопка «Собрать план» → лоадер с ИИ-текстом → **превью плана** (цели + задачи с временем/важностью/
  дедлайном) → правка/принять → запись в Drift + синк. Не автосейв.
- Фича **визуально выделена** как «магия триала», чтобы сразу попробовали (твоё требование).

## 3. Что ИИ возвращает (контракт `POST /api/v1/ai/onboarding-plan`)

Вход: `{ answers: string, date, timezone, locale }`. Выход (Zod-провалидирован, provider-agnostic,
thinkingBudget:0, языковая directive):
```
{
  goals:  [{ title, horizon? }],
  tasks:  [{ title, type, priority, scheduledAt?, deadline?, durationMinutes?, note? }],
  food_prefs?: { tracks_food, tracks_water, tracks_sleep }
}
```
`deadline` при записи → `type='deadline'` + `scheduledAt`=срок (дефолт C). `color`/`location` — локальные поля.

Кнопка быстрого добавления `POST /api/v1/ai/quick-add`: вход `{ text, date, timezone, locale }` →
одна структурированная задача (тип/важность/цвет/время/место), превью → подтверждение. Regex-парсер
`nl_datetime.dart` остаётся как быстрый предпроход; ИИ добавляет понимание свободной речи.
