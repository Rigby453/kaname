- **[ ] Backend AI-эндпоинт** `POST /api/v1/ai/onboarding-plan` (premium-gated): вход = ответы юзера +
  дата/таймзона; выход = структурированный план (goals[], tasks[] с priority/scheduledAt/deadline,
  опц. food_prefs). JSON-схема, thinkingBudget:0, языковая directive (уже есть), провайдер-агностик.
- **[ ] Применение плана:** превью-экран «вот твой план» → пользователь принимает/правит → задачи и
  цели создаются в Drift + синк. Не автосохранять молча — подтверждение (как утренний разбор).
- **[ ] Выделение фичи:** бейдж/подсветка «Premium trial», call-to-action чтобы попробовали сразу.
- **Вопросы онбординга (черновик, на утверждение):**
