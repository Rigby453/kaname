# Kaname

### Планировщик дня, который помогает не потерять главное.

Большинство планировщиков помогают **записывать задачи**.  
**Kaname помогает принимать решения**, когда день идёт не по плану.

[Лендинг](https://rigby453.github.io/kaname/) •

## Почему появился Kaname

Практически любой планировщик позволяет быстро записывать задачи.

Настоящая проблема появляется позже.

Когда день неожиданно меняется, приходится самостоятельно решать:

- что действительно важно сегодня;
- что можно перенести;
- чем можно пожертвовать;
- как не потерять задачу, ради которой вообще открывался планировщик.

Именно в этот момент большинство приложений перестают помогать.

Kaname создавался вокруг одной идеи:

> **Помочь сделать главное, а не попытаться успеть всё.**

Поэтому основой приложения стали **Главные задачи**.

Каждый день можно выбрать до трёх задач, которые действительно должны быть выполнены. Остальной день перестраивается вокруг них.

---

## Чем отличается

| Обычный планировщик | Kaname |
| :------------------ | :----- |
| Все задачи выглядят одинаково важными | До трёх задач становятся **Главными** |
| После неудачного дня всё приходится переносить вручную | Утром приложение предлагает новый план |
| Любая задача может незаметно потеряться | Главное всегда остаётся в центре внимания |
| Через месяц непонятно, почему планы постоянно срываются | Дневник помогает увидеть реальные причины |

# Как проходит день

## 🌅 Утро

Вместо списка просроченных задач Kaname сразу показывает готовый вариант сегодняшнего дня.

Если вчера что-то осталось незавершённым, приложение предлагает, куда лучше перенести задачи, чтобы сохранить баланс и не потерять главное.

Можно:

- принять весь план;
- изменить отдельные задачи;
- полностью собрать расписание самостоятельно.

---

## ⭐ Главное

Главные задачи — не просто цветная метка.

Они вынесены в отдельный блок интерфейса и становятся центром дня.

Именно вокруг них строится всё остальное.

Если планы приходится менять, Kaname старается сохранить именно эти задачи.

---

## 📅 Один план на весь день

В одном расписании находятся:

- задачи;
- события;
- учёба;
- питание;
- тренировки;
- сон;
- фокус-сессии.

Все элементы находятся в одной временной линии и свободно перемещаются простым перетаскиванием.

Ничего не разбросано по разным приложениям.

---

## 🌙 Вечер

Перед сном Kaname предлагает план следующего дня.

Его можно принять одним нажатием, изменить вручную или полностью перестроить.

Следующее утро начинается уже с готового плана.

---

## 📖 Дневник

В конце дня можно отметить настроение, оценить день и указать, что помешало выполнить главное.

Например:

- усталость;
- соцсети;
- неожиданно появившиеся дела;
- неправильная оценка времени;
- собственная причина.

Со временем становится видно не один неудачный день, а закономерности, которые действительно влияют на продуктивность.
---

# Возможности

Kaname не пытается заменить десяток разных приложений.

Он объединяет всё, что занимает время в течение дня, в одном плане и помогает сохранять фокус на главном.

### 📌 Планирование

- до трёх Главных задач на день;
- утренний и вечерний разбор;
- автоматический перенос незавершённых задач;
- календарь и расписание;
- повторяющиеся задачи;
- свободное перетаскивание задач;
- теги и поиск;
- блочный редактор заметок.

---

### 🥗 Питание

Питание является частью общего расписания, а не отдельным модулем.

Можно добавлять продукты:

- вручную;
- голосом;
- по фотографии;
- по штрихкоду;
- по рецепту;
- из меню ресторана.

ИИ способен собрать рацион с учётом заданных КБЖУ и предпочтений пользователя.

---

### 🏋️ Тренировки

Тренировки планируются так же, как и остальные события дня.

Доступны:

- программы тренировок;
- недельные и месячные циклы;
- режим тренера;
- таймер упражнений;
- описание техники выполнения.

---

### 🌙 Здоровье

При необходимости можно включить дополнительные разделы:

- сон;
- вода;
- дыхательные практики;
- медитации.

По умолчанию интерфейс остаётся максимально простым и не перегруженным.

---

### 🎯 Фокус

Встроенные режимы концентрации:

- 25 / 5;
- 50 / 10;
- 90 / 20;
- фирменный режим **67 / 15**.

Фокус-сессии становятся частью общего расписания дня.

---

### 🎯 Долгосрочные цели

Цели можно планировать на любой горизонт — от месяца до нескольких лет.

Следующий шаг автоматически превращается в сегодняшнюю задачу.

---

### 🔥 Стрики

Небольшая система мотивации помогает сохранять регулярность.

Есть:

- серия успешных дней;
- заморозки стрика;
- карточка для публикации прогресса.

Геймификация намеренно остаётся лёгкой и не отвлекает от главной задачи — выполнения действительно важных дел.

---

### 🐻 Kai

Kai — небольшой маскот приложения.

Он сопровождает утренние и вечерние разборы, помогает на пустых экранах и делает взаимодействие с приложением немного живее, оставаясь ненавязчивым.

---

# ИИ — там, где он действительно полезен

Kaname остаётся полностью рабочим приложением даже без искусственного интеллекта.

Все основные сценарии доступны бесплатно.

Premium не открывает новые разделы и не ограничивает бесплатную версию. Он делает существующие функции более персонализированными.

| Бесплатно | Premium — $10/мес |
| :--------- | :---------------- |
| Утренний и вечерний разбор по правилам | Несколько вариантов плана с учётом привычек пользователя |
| Перенос задач | Более реалистичное распределение нагрузки |
| Главные задачи | Глубокий анализ причин переносов |
| Дневник | Персональные рекомендации и подробный разбор |
| Оффлайн-синхронизация | ✓ |
| Без рекламы | ✓ |

Все обращения к ИИ выполняются только через backend.

Ключи API никогда не попадают в клиентское приложение.

Поддерживаются Claude и Gemini через настройку `.env`.

---

# Без рекламы

В Kaname нет рекламы.

Ни в бесплатной версии, ни в Premium.

Это осознанное решение.

Планировщик должен помогать сосредоточиться на своих задачах, а не конкурировать за внимание пользователя.
