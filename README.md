# Kaname

### Планировщик дня, который помогает не потерять главное.

Большинство планировщиков помогают **хранить задачи**.

**Kaname помогает сохранить действительно важное.**

---

> **Главная идея Kaname — не помочь успеть всё, а помочь не потерять главное.**

Большинство планировщиков через несколько недель превращаются в длинный список дел. Когда день идёт не по плану, приходится самостоятельно решать, что перенести, чем пожертвовать и что действительно важно.

Именно здесь Kaname начинает помогать.

Вместо бесконечного списка задач каждый день можно выбрать **до трёх Главных задач**. Всё остальное — учёба, встречи, питание, тренировки, сон и остальные дела — становится частью одного общего плана и подстраивается вокруг них.

---

# Почему Kaname

| Обычный планировщик | Kaname |
| :------------------ | :------ |
| Все задачи выглядят одинаково важными | До трёх задач становятся **Главными** |
| Один сорванный день ломает весь план | Утром приложение помогает спокойно пересобрать день |
| Вечером приходится переносить задачи вручную | Уже готов новый вариант плана |
| Через месяц непонятно, почему планы постоянно срываются | Дневник помогает увидеть реальные причины |

> **Не пытаться сделать всё. Помочь сделать главное.**

---

# Как проходит день

## 🌅 Утренний разбор

Вместо списка просроченных задач Kaname сразу показывает подготовленный план.

Если вчера что-то осталось незавершённым, приложение предлагает, куда лучше перенести задачи. План можно принять полностью, изменить отдельные элементы или собрать самостоятельно.

---

## ⭐ Главное

Главные задачи — центр приложения.

Это не просто цветная метка в длинном списке.

Они вынесены в отдельный блок интерфейса и получают максимальный приоритет. Если планы приходится менять, Kaname в первую очередь старается сохранить именно их.

---

## 📅 Один план на весь день

В одном расписании находятся:

- задачи;
- события;
- учёба;
- питание;
- тренировки;
- сон;
- фокус-сессии.

Все элементы находятся в одной временной линии и свободно перемещаются простым перетаскиванием.

Ничего не разбросано по разным приложениям.

---

## 🌙 Вечерний разбор

Перед сном Kaname предлагает готовый план следующего дня.

Его можно принять одним нажатием, изменить вручную или полностью перестроить самостоятельно.

Следующее утро начинается уже с готового плана.

---

## 📖 Дневник

В конце дня можно отметить настроение, оценить результат и указать, что помешало выполнить главное.

Со временем становится видно не один неудачный день, а реальные причины, из-за которых планы регулярно срываются.

---

# Всё остальное — часть того же дня

День состоит не только из задач.

Нужно успеть поесть, сходить на тренировку, подготовиться к экзамену, оставить время на отдых.

Поэтому питание, тренировки, сон, фокус-сессии и долгосрочные цели становятся такими же событиями дня, как пара в университете или дедлайн.

> Всё, что занимает время, должно участвовать в планировании.

Экран **Health** остаётся максимально простым.

По умолчанию отображаются только вода и сон. Остальные разделы включаются только тогда, когда действительно нужны.

# Возможности

## 📌 Планирование

Основа Kaname — управление самым важным.

- до трёх Главных задач на день;
- утренний и вечерний разбор;
- автоматический перенос незавершённых задач;
- календарь и расписание;
- повторяющиеся задачи;
- свободное перетаскивание задач;
- теги и поиск;
- заметки с блочным редактором.

---

## 🎯 Фокус

Kaname помогает не только составить план, но и начать выполнять его.

Доступны режимы концентрации:

- 25 / 5;
- 50 / 10;
- 90 / 20;
- фирменный режим **67 / 15**.

Фокус-сессии становятся частью общего расписания и учитываются при планировании дня.

---

## 🥗 Питание

Питание находится внутри общего плана, а не в отдельном приложении.

Можно добавлять еду:

- вручную;
- голосом;
- по фотографии;
- по штрихкоду;
- по рецепту;
- из меню ресторана.

При необходимости приложение помогает поддерживать баланс КБЖУ без жёстких ограничений.

---

## 🏋️ Тренировки

Тренировки становятся частью расписания.

Доступны:

- программы на неделю и месяц;
- циклы тренировок;
- запуск прямо из плана;
- режим тренера;
- таймеры;
- описание техники упражнений.

---

## 🌿 Сон и восстановление

Kaname учитывает не только работу, но и восстановление.

Можно отслеживать:

- сон;
- воду;
- дыхательные практики;
- медитации.

---

## 🎯 Долгосрочные цели

Большие цели превращаются в конкретные действия.

Можно планировать цели на месяц, год или несколько лет.

Следующий шаг автоматически становится задачей сегодняшнего дня.

---

## 🔥 Стрики

Небольшая система мотивации помогает сохранять регулярность.

Есть:

- серии успешных дней;
- заморозки;
- карточка прогресса для публикации.

Геймификация остаётся дополнительным инструментом и не становится главной целью приложения.

---

## 🐻 Kai

Kai — небольшой маскот Kaname.

Он сопровождает утренние и вечерние разборы, помогает на пустых экранах и делает взаимодействие с приложением более живым.

Без лишних настроек и сложных механик — только несколько эмоций и спокойная помощь.

---

# ИИ помогает принимать решения, а не заменяет человека

Kaname остаётся полноценным планировщиком даже без искусственного интеллекта.

Все основные функции доступны бесплатно и работают по понятным правилам.

Premium делает эти же функции более персональными.

| Бесплатно | Premium — $10/мес |
| :-------- | :---------------- |
| Утренний и вечерний разбор по правилам | Несколько вариантов плана |
| Перенос задач | Умное перераспределение нагрузки |
| Главные задачи | Анализ причин переносов |
| Дневник | Более глубокие рекомендации |
| Оффлайн-синхронизация | ✓ |
| Без рекламы | ✓ |

ИИ работает только через backend.

API-ключи никогда не попадают в клиентское приложение.

Провайдер модели выбирается через `.env`:

- Claude;
- Gemini.
